## Pet-проекты

[anarchy_bot](https://github.com/gmankab/anarchy_bot) - телеграм бот, который раздает админки, выдает муты по голосованию

[tg_api_gen](https://github.com/gmankab/tg_api_gen) - библиотека, которая генерирует реалистичные device_model, app_version и system_version (android и ios) для телеграм юзерботов

[reposter](https://github.com/gmankab/reposter) - скрипт для пересылки сообщений из одного тг канала в другой в реальном времени, сохраняет историю редактирования сообщений в комментариях, отмечает сообщение как удаленное, если оно было удалено, а также уведомляет, когда в канале начинается стрим. Поддерживается отправка сообщений с несколькими вложениями. Простая, интуитивно понятная настройка и запуск

[yt_stream_recorder](https://github.com/gmankab/yt_stream_recorder) - скрипт для записи YouTube стримов. Запускается 1 раз, записывает все стримы на выбранном канале

[suggest](https://github.com/gmankab/suggest) - тг бот с предложкой. Есть возможность забанить пользователя, чтобы он больше не мог предлагать посты,rвыбрать время, на которое пользователь будет забанен. Поддерживается отправка сообщений с несколькими вложениями

[PrettyGit](https://github.com/gmankab/PrettyGit) - удобный интерфейс для git, поддерживает коммиты, пуши и загрузку на [pypi.org](https://pypi.org)

[easyselect](https://github.com/gmankab/easyselect) - библиотека, позволяющая пользователю выбирать между несколькими элементами в консоли с помощью клавиатуры. Поддерживает очень длинные списки, которые не помещаются на экран, [стили rich](https://rich.readthedocs.io/en/stable/style.html), управление кнопками `up`, `down`, `left`, `right`, `wasd`, `j`, `k`, `home`, `end`, `page up`, `page down`
